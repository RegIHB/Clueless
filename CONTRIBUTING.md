# Contributing

Thanks for contributing to Clueless.

## Design source

- Figma Make source: [Clueless app landing page](https://www.figma.com/design/3HtSDiVvGA6xZadJuRTuZs/Clueless-app-landing-page)

## Pull request checklist

- Keep PRs focused and small.
- Include a short summary of what changed and why.
- Run local checks before requesting review:
  - `pnpm lint`
  - `pnpm build`
- If UI behavior changed, add screenshots or a short video.
- Ensure required CI checks pass before merge.
